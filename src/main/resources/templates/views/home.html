Get started

Go to deployments and click on to deploy a new process and other resources. Or, use the Camunda modeler or a Zeebe client.

Select the deployed process, look for the start event and click on to create a new process instance.

Drive the process instance

Select a task and click on to complete it.

Or, select a start event or a catch event and publish a message or manipulate the time to trigger the event.

Demo process

Try out Zeebe Play by using a demo process. Click on the buttons below.

  • First, deploy the demo process
  • Then, create a new process instance
User forms

Select a user task with a Camunda form and click on to see the form and fill it out. The user task gets completed with the form data.

Connectors New

Select a task of an available connector and click on to invoke the connector and complete the task.

Go to connectors and see the available connectors.

Rewind New

Select a completed task and click on to rewind the process to this task. From here, you can try out different paths or inputs.
